# Reproducibilidad

## Trastornos Neurocognitivos y Demencia en América Latina e Iberoamérica (2000-2025)

### Requisitos
- Python 3.8+, pandas, matplotlib, seaborn

### Pasos
1. `git clone https://github.com/juanmoisesd/trastornos-neurocognitivos-demencia-america-latina-2000-2025`
2. `pip install -r requirements.txt`
3. Ejecutar `analysis.ipynb`
