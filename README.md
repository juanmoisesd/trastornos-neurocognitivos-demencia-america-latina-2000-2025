# Trastornos Neurocognitivos y Demencia en América Latina e Iberoamérica (2000-2025)

## Descripción

Base de datos epidemiológica sobre trastornos neurocognitivos y demencia en América Latina e Iberoamérica (2000-2025). Incluye prevalencia, incidencia, factores de riesgo, acceso a diagnóstico y tendencias temporales.

## Estructura

- `prevalencia_demencia.csv` — Tasas de prevalencia por país y año
- `factores_riesgo.csv` — Factores de riesgo epidemiológicos
- `acceso_diagnostico.csv` — Disponibilidad de diagnóstico y tratamiento
- `tendencias_temporales.csv` — Evolución 2000-2025
- `analysis.ipynb` — Análisis reproducible en Python
